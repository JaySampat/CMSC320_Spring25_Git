Q1: Why might we consider using Git?

Answer: We might consider using Git in order to save our code somewhere that is not our local machine. Additionally, we can consider using Git to be able to collaborate with other on projects and tasks.

Q3: What is the difference between Merging and Rebasing?

Answer: Merging takes two branches and combines them. Rebasing moves the changes to the end of the other branch.
